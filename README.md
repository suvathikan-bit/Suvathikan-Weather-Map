Overview

This is a professional interactive global weather map built with Leaflet.js and OpenWeatherMap API, designed for all weather enthusiasts and developers. The dashboard allows users to explore weather information anywhere on Earth, including cities, villages, and streets.

Click anywhere on the map to see current weather including:

Temperature

Feels Like

Clouds

Humidity

Wind Speed

Weather Condition

Features

Google Maps-style layers: Street, Satellite, Dark, Light, Terrain

Switching map layers

Changing temperature units (¬∞C / ¬∞F)

Toggling logo visibility

Weather info popup on map click

Full world map coverage: all cities, villages, streets

Modern arranged UI for a professional look

Your logo and contact info displayed in settings

Tech Stack

HTML, CSS, JavaScript

Leaflet.js for interactive maps

OpenWeatherMap API for real-time weather

Fully client-side, no backend required

How to Run

Clone this repository.

Ensure logo.png is in the project folder.

Open the index.html in any modern browser.

Click anywhere on the map to see the weather info.

Use the settings panel to switch layers, change units, or toggle logo visibility.

Notes

API key for OpenWeatherMap is already included (3729667eb86355595d2be48f76ccc8b0) for demonstration. Replace it with your own key for production.
